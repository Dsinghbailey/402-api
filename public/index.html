402 API

A collection of demo api tools using the 402 payment protocol

Quickstart (Requires a wallet with Base USDC)

Step 1: Install required dependencies

npm install viem x402-fetch dotenv

Step 2: Use the following code to make API calls

import { privateKeyToAccount } from "viem/accounts";
import { wrapFetchWithPayment, decodeXPaymentResponse } from "x402-fetch";
import dotenv from "dotenv";

dotenv.config();

// Set your private key in .env file
const account = privateKeyToAccount(process.env.PRIVATE_KEY);
const fetchWithPayment = wrapFetchWithPayment(fetch, account);

// Example API call
const url = "http://402api.com/text-to-image?prompt=a%20blue%20ball";

fetchWithPayment(url, {
  method: "GET",
})
  .then(async (response) => {
    const body = await response.json();
    console.log(body);
  })
  .catch((error) => {
    console.error(error.response?.data?.error);
  });

Available API Endpoints (GET or POST)

/text-to-image$0.10

Description: Generates an image based on a text prompt using OpenAI's DALL-E 3 model.

Parameters:

  • prompt (query parameter): The text prompt for image generation. (e.g., "a beautiful landscape")

Example Return:

{
    "image": "https://oaidalleapiprodscus.blob.core.windows.net/...",
    "prompt": "a beautiful landscape"
}

Try it out:

/word-count$0.01

Description: Counts the number of words in a given text.

Parameters:

  • text (request body): The text for which to count words.

Example Return:

{
    "wordCount": 5,
    "text": "This is a sample text."
}

Try it out:

/sentiment-analysis$0.05

Description: Performs basic sentiment analysis on the provided text, classifying it as positive, negative, or neutral.

Parameters:

  • text (request body): The text to analyze.

Example Return:

{
    "sentiment": "positive",
    "text": "This is a happy and wonderful day!",
    "scores": {
        "positive": 3,
        "negative": 0
    }
}
